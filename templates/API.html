Use the api get instructional videos

Send GET or POST data to http://127.0.0.1:8000/api/getvideo. The format of the data is json data.
The sample json format sent is:
{ "gender": "Male", "age": 20, "height": 170, "weight": 60, }
The value of the gender field needs to be male or female and is not case-sensitive.
The value of the age field needs to be an integer.
The value of the height field needs to be an integer and unit is cm.
The value of the weight field needs to be an integer and unit is cm.

If you would like to send and receive data through the page, please click on the button below to jump.
